# 👩‍💻 Kübra Durak – Kişisel Portfolyo Web Sitesi

Bu proje, ASP.NET MVC 5 mimarisi kullanılarak geliştirilmiş bir kişisel portfolyo web sitesidir.  
Hakkımda bilgisi, yetenekler, projeler, referanslar ve iletişim gibi bölümleri dinamik olarak veri tabanından çekerek gösterir.

---

## 🚀 Özellikler

- **Hakkımda** sayfası (veritabanından dinamik veri)
- **Projeler** bölümü (SQL Server üzerinden yönetilir)
- **Referanslar/Testimonial** listesi
- **İletişim Bilgileri** bölümü
- **CV indir** butonu (PDF dosyası)
- Modern ve sade responsive tasarım
- Mobil uyumlu Bootstrap arayüz
- Admin panel olmadan SQL ile kolay içerik yönetimi

---

## 🛠️ Kullanılan Teknolojiler

- ASP.NET MVC 5 (.NET Framework)
- Entity Framework 6 (Code First)
- SQL Server / LocalDb
- Bootstrap 5
- Razor View Engine
- Visual Studio 2022
